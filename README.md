# QC検定2級 勉強ログ（GitHub × 学習管理）

## 📌 このリポジトリについて
このリポジトリは、QC検定2級の学習内容を  
**GitHubを使って整理・記録・振り返り** するための個人用学習ログです。

目的は以下の3つです。

- QC検定2級に **一発合格** する
- 統計的考え方を「理解ベース」で身につける
- Git / GitHub に自然に慣れる

※ 点数管理や暗記ではなく  
**「どこで詰まり、どう克服したか」** を重視します。

---

## 🎯 学習方針（重要）
- 正規分布は **必ず標準正規分布表とセットで考える**
- 計算よりも **考え方・判断の流れ** を優先
- 間違いは「失敗」ではなく **資産** として残す
- 同じ過去問を **100点取れるまで繰り返す**
