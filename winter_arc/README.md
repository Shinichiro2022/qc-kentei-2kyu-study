# Winter Arc Operating System

## 基本思想
- 冬は「気合」ではなく「設計」で乗り切る
- 感情に左右されず、行動が先に立ち上がる仕組みを作る
- 小さくても「毎日、前に進んだ証拠」を Git に残す

---

## Time Shift（ウィンタータイム）

### ルール
- 生活時間を **通常より2時間早める**
- その他大勢が動く前に、すでに動いている状態を作る

### 目的
- 静かな時間を確保する
- 比較・ノイズ・割り込みを避ける
- 思考と学習を「先取り」する

※ 完璧に2時間守れなくてもOK  
※ 早めた時間の一部を使えたら勝ち

---

## Fast Routine（即動ルール）

### 方針
- 考える前に動く
- 判断コストを限りなくゼロにする
- 行動の初速を固定する

### Daily Fast Routine
- PCを開く
- Gitリポジトリを開く
- `journal.md` または `progress.md` を **1行更新**
- そのまま commit

→ 内容の質は問わない  
→ 書いた瞬間に「成功」とする

---

## Core Habits（毎日の柱）

### 1. 運動 × 勉強
- 軽く体を動かしてから学習に入る
- 身体 → 思考 の順番を守る

### 2. 7日間運動
- 連続7日を1セットとする
- 内容・強度は問わない
- 「続けた事実」を最優先する

---

## Night Routine（就寝前）

### ナイトジャーナル
- 感謝を書き出す（小さなことでOK）
- 今日の気づき・アイディアを記録する
- 反省・自己否定は禁止

### Tomorrow Setup（前日仕込み）

- 寝る前に「明日やること」を1〜3個だけ書く
- 内容は小さくて具体的にする
- 判断は夜に終わらせ、朝は実行だけにする

例：
- QC検定：正規分布 逆引き問題 3問
- 運動：散歩10分
- Git：journal.md を1行更新

※ 全部できなくてもOK  
※ 1つ実行できたら勝ち
イゴールは15分単位でスケジュール
https://youtu.be/YoORq3erAA0?si=LRdvXh1NPNTvFrUC

### 10分瞑想
- 呼吸に集中するだけでOK
- 評価・成果は求めない

### 30分読書
- 分野は自由
- インプットの質より「静かな時間」を重視

---

## 勝ちの定義（重要）

以下のうち **1つでも達成できたら勝ち** とする。

- 2時間早めて動けた
- ファストルーティンを実行した
- 運動 or 勉強をした
- Git に何かを残した
- 夜のジャーナルを書いた

→ 完璧な日は不要  
→ 止まらなかった日が勝ち

---

## この Winter Arc のゴール
- 行動が習慣として固定されている
- 学習と生活が「構造」で回っている
- 冬を越えた証拠が Git に残っている

## Time Control（15分単位のスケジューリング）

### 方針
- 1日を **15分単位** で扱う
- 長時間の計画は立てない
- 「次の15分に何をやるか」だけを決める

### 考え方
- 人生は時間の使い方でほぼ決まる
- 集中力は30分も続かない
- 15分なら誰でも始められる

### ルール
- 15分 × 1コマでも前進扱い
- 空いた15分は「運動・勉強・記録」に充てる
- 完璧なスケジュールは不要

→ 15分を制する者が1日を制する

---

## Why Different（なぜ人と違う行動を取るのか）

### 信念
- 誰もやらないことをやるしか、人生で勝つ方法はない
- 多くのことを成し遂げるには、人と同じ行動では足りない
- 「普通の努力」では、普通の結果しか得られない

### Winter Arc における選択
- 時間を2時間早める
- 迷う前に動く（ファストルーティン）
- 運動・学習・内省を習慣化する
- Git に記録を残し、逃げ道をなくす

これらはすべて  
**多数派がやらない行動** であり、  
だからこそ意味がある。

---

## 自分へのメッセージ（折れそうな時に読む）

- 今やっていることは地味で、誰にも褒められない
- でも、後から効くことだけを選んでいる
- 周りが動いていない時に動けている

→ それだけで、もう違う道に入っている

---

## 勝ちの再定義

- 派手な成果は不要
- 連続した15分が積み上がっていればOK
- 今日も「人と違う行動」を1つ取れたら勝ち

Winter Arc は  
短距離走ではなく、  
**静かに差が開く長期戦**。
