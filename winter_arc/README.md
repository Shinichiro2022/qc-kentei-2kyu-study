# Winter Arc Operating System

## 基本思想
- 冬は「気合」ではなく「設計」で乗り切る
- 感情に左右されず、行動が先に立ち上がる仕組みを作る
- 小さくても「毎日、前に進んだ証拠」を Git に残す

---

## Time Shift（ウィンタータイム）

### ルール
- 生活時間を **通常より2時間早める**
- その他大勢が動く前に、すでに動いている状態を作る

### 目的
- 静かな時間を確保する
- 比較・ノイズ・割り込みを避ける
- 思考と学習を「先取り」する

※ 完璧に2時間守れなくてもOK  
※ 早めた時間の一部を使えたら勝ち

---

## Fast Routine（即動ルール）

### 方針
- 考える前に動く
- 判断コストを限りなくゼロにする
- 行動の初速を固定する

### Daily Fast Routine
- PCを開く
- Gitリポジトリを開く
- `journal.md` または `progress.md` を **1行更新**
- そのまま commit

→ 内容の質は問わない  
→ 書いた瞬間に「成功」とする

---

## Core Habits（毎日の柱）

### 1. 運動 × 勉強
- 軽く体を動かしてから学習に入る
- 身体 → 思考 の順番を守る

### 2. 7日間運動
- 連続7日を1セットとする
- 内容・強度は問わない
- 「続けた事実」を最優先する

---

## Night Routine（就寝前）

### ナイトジャーナル
- 感謝を書き出す（小さなことでOK）
- 今日の気づき・アイディアを記録する
- 反省・自己否定は禁止

### 10分瞑想
- 呼吸に集中するだけでOK
- 評価・成果は求めない

### 30分読書
- 分野は自由
- インプットの質より「静かな時間」を重視

---

## 勝ちの定義（重要）

以下のうち **1つでも達成できたら勝ち** とする。

- 2時間早めて動けた
- ファストルーティンを実行した
- 運動 or 勉強をした
- Git に何かを残した
- 夜のジャーナルを書いた

→ 完璧な日は不要  
→ 止まらなかった日が勝ち

---

## この Winter Arc のゴール
- 行動が習慣として固定されている
- 学習と生活が「構造」で回っている
- 冬を越えた証拠が Git に残っている
